
# slt-js Monorepo

Monorepo containing the slt-js ecosystem:
- slt-js-core: reference CLI transpiler
- slt-acorn-plugin: Acorn plugin to recognize localized keywords (simple reference)
- slt-babel-plugin: Babel plugin to perform keyword -> canonical AST transform and forward transpile
- slt-cli: a thin CLI that wires plugins together
- language packs: @slt-lang/es, fr, it, zu
- slt-syntax-vim: syntax file for Vim
- web-demo: lightweight demo site (placeholder)

This archive is generated as a starting point. Use `pnpm` or `npm` workspaces to develop.
